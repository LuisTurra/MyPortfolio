Desafio do Caminho de Código

Bem-vindo ao Desafio do Caminho de Código! Clique nos botões "Cima", "Baixo", "Esquerda" ou "Direita" para criar uma sequência de comandos que leve o cubo azul do ponto A (verde) ao ponto B (vermelho) pelo caminho branco. Clique em "Iniciar" para mover o cubo. Se a sequência estiver correta, você alcançará o objetivo! Clique em "Reiniciar" para tentar um novo caminho.
