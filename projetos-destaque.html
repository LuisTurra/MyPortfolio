Detecção de Anomalias em Transações Bitcoin

Desenvolvi um pipeline completo de detecção de anomalias em transações do blockchain Bitcoin usando apenas recursos gratuitos do Google Cloud Platform. Com dbt, criei models de staging e feature engineering para transformar dados públicos do BigQuery em atributos relevantes como valor da transação, contagem de inputs/outputs e razão da taxa. Implementei uma solução rule-based avançada com thresholds dinâmicos e anomaly score composto devido às limitações do Sandbox. O resultado é um sistema explicável e reproduzível para identificar padrões suspeitos como mixing e transações de alto risco.

Previsão de Taxas de Câmbio com RNN

Sistema de previsão de taxas de câmbio, focando em pares como USD/BRL, utilizando modelos de redes neurais recorrentes como LSTM, GRU e variantes para analisar séries temporais financeiras. Os dados são obtidos via API Alpha Vantage com cache local, pré-processados com normalização e sequências temporais, e treinados com TensorFlow/Keras para comparar desempenhos via MSE, selecionando o melhor modelo. Uma interface web interativa é criada com Streamlit, permitindo seleção de moedas, previsão de até 90 dias e visualizações dinâmicas com Plotly. No geral, demonstra aplicação de ML em finanças, com salvamento de modelos e relatórios visuais para análise comparativa.
