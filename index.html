Sistema de Recomendação de Cultura

Projeto pessoal em que foram desenvolvidos dois sistemas de recomendação de colheita. O primeiro utiliza valores de mínimo, máximo e média, usando uma base de dados de referência, para recomendar o cultivo de cada tipo de cultura selecionada. O segundo sistema emprega o algoritmo de machine learning Floresta Aleatória e simulação de APIs para recomendar culturas com base nas propriedades do solo, localização e clima.
